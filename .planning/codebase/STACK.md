# Codebase Tech Stack

## Core Technologies
- **Language**: JavaScript (ES6+), React 19
- **Runtime**: Node.js (development), Browser (production)
- **Framework**: React (Client-side rendering)
- **Build Tool**: Vite 8
- **Styling**: Tailwind CSS 3 / 4 via PostCSS

## Key Dependencies
### Frontend
- `react`, `react-dom`: v19.2.5
- `react-router-dom`: v7.14.2 (Routing)
- `framer-motion`: v12.38.0 (Animations)
- `lucide-react`: v1.8.0 (Icons)

### Development
- `vite`: Build tool and dev server
- `eslint`: Linting
- `tailwindcss`, `postcss`, `autoprefixer`: CSS styling

## Configuration Files
- `package.json`: Dependency management
- `vite.config.js`: Vite server and build configuration
- `tailwind.config.js` / `postcss.config.js`: Styling configuration
- `eslint.config.js`: Linting rules
